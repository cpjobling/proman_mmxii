p-2011-239 – Optical Wireless Communication for Indoors

Set by Dr Karin Ennser ( k.ennser@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Available for selection in next round? No

This project has been allocated to student with id: 556379

Description

The project is related to emerging technology on indoor optical wireless communciation. Several components and system technologies will be investigated. Potential applications for daily life will be assessed for different applications as sensor, data/video transmission and others.

Cross disciplinary research theme(s): Aerospace and manufacturing

Special requirements

Project may need a small budget for small items.
