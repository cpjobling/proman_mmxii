p-2011-001 – Mobile Apps for Engineering

Set by Dr Chris Jobling ( c.p.jobling@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Available for selection in next round? No

Description

The Head of Teaching in the College of Engineering is keen to introduce innovation into the courses in Engineering. One of his ideas is to develop a suite of learning activities that could be delivered as Apps on smart phones or tablets. This project will explore the market-place for such applications and develop examples using either the iOs, Android or Web App approach. As an extreme example of the sort of thing that is possible, you might want to check out the Mobile and Tablet apps that are available for Wolfram|Alpha (see http://products.wolframalpha.com/mobile/). Our ambitions would be more limited, but this is an opportunity to use your ICCT skills and try out app development.

Special requirements

For iPhone/iPad development, a Mac will be required: It is unlikely to be supplied!

p-2011-002 – Proman Olympic

Set by Dr Chris Jobling ( c.p.jobling@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Available for selection in next round? No

This project has been allocated to student with id: 526528

Description

Some time ago, an ICCT student developed a web application called Proman that was used during the project allocation process a couple of years ago. Since then, for various reasons, project allocation has moved over to Blackboard. However, in the meantime, developments in web applications technology have moved on considerably. This project will revisit Proman and develop a new project management web app using either Drupal 7 or Ruby on Rails 3. New features worth considering would be the introduction of social networking and peer support features, mobile notifications, etc. An interesting and challenging project for what will be the year of the London Olympics.

Special requirements

Access to a web server with the domain proman.swan.ac.uk is already available.

p-2011-003 – Social Engineering

Set by Dr Chris Jobling ( c.p.jobling@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Available for selection in next round? No

This project has been allocated to student with id: 555069

Description

This project will explore the possibilities of using web-based and mobile tools for social networking for building a community in the College of Engineering. A comprehensive survey of existing social networks such as Facebook and Twitter, social tools such as grou.ps and Ning, the Blackboard community pages and self-hosted open-source options such as Moodle, Elgg, Buddypress, and Drupal will be undertaken. The idea will be to find the best platform with which to build an on-line community within the College of Engineering that will be attractive to both the learners and teachers. This project is a follow-on to an earlier project that explored the use of social networking in peer support.

Special requirements

Hosting for self-hosted services is available in the form of a LAMP server.

p-2011-181 – Spearphishing

Set by Prof Thomas Chen ( t.m.chen@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Available for selection in next round? No

Description

Spearphishing is a form of phishing and social engineering that aims to deceive a computer user with convincing, personalized email. It is the first stage of an attack to obtain valuable information such as passwords, and has been increasing in popularity in successful attacks that have led to costly data theft for companies and organizations. The project is to (a) experimentally investigate how to harvest personal information from the web about potential targets (b) experimentally measure the effectiveness of spearphishing and other social engineering attacks to obtain valuable personal information.

Special requirements

PC with access to campus network.

p-2011-182 – Biology-Inspired Networking

Set by Prof Thomas Chen ( t.m.chen@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Available for selection in next round? No

This project has been allocated to student with id: 583406

Description

Biological systems often demonstrate highly advantageous characteristics, such as self healing, self organization, adaptation, and self learning. Researchers are seeking to apply lessons from biology to improve the design of communication networks. For example, ants are adept at finding the shortest routes to food, which might offer lessons for network routing protocols. The way that the human body heals itself and fights off infections may offer insights for self-healing and secure networks. The project will choose one notable example from biology and demonstrate how biology can lead to improvements in current network design or operation. The improvement should be demonstrated mathematically or by computer simulations.

Special requirements

Access to network simulation software, e.g., OPNET, OMNET.

p-2011-183 – Blackhat Search Engine Optimization

Set by Prof Thomas Chen ( t.m.chen@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Available for selection in next round? No

This project has been allocated to student with id: 620417

Description

Google's search engine presents the results of a search according to a ranking of relevant web pages. The placement of a company's web page near the top of search results can mean a big difference in business. Companies use SEO (search engine optimization) techniques to increase their ranking in Google, but sometimes veer into "blackhat SEO" techniques that Google considers to be unethical. This project aims to (a) review the algorithms used by Google to rank web pages (b) review whitehat and blackhat SEO techniques (c) experimentally measure the use of blackhat SEO techniques on the web.

Special requirements

PC with web access.
