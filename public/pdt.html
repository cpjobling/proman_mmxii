Project Descriptions

p-2011-277 – Design of Portable Bio-Sensor System for Drug Testing Using SMART PHONE Technology

Set by Dr Paul Holland ( p.m.holland@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

The student will study a specification for a Bio-Sensing product and design a system to implement the specification. The student will identify the basic building blocks and what functionality they will contain. Particular focus will be to design a system in modular form using known existing standards and protocols. The student will also design a new package type with a disposable breakable lid to allow a blood doplet to be applied to the sensor.

Cross disciplinary research theme(s): Health and Sport
