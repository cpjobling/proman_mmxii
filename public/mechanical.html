p-2011-235 – Modelling Cable Coating flow – applications in the polymer industry

Set by Prof Michael Webster ( m.f.webster@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

This project aims to develop the modelling capability to aid in the prediction of polymer coatings in industry, consistently and with minimal residual stressing on continuous cable-coating processing lines at low line speeds. The study will integrate with existing research in the department, advances in which could prove vital in ensuring modern industry remains viable and competitive. The project will focus on Matlab programming and model examples.

Special requirements

Mathlab, Personal Laptop (would be helpful)

p-2011-238 – Rheological material functions and user-interfaces

Set by Prof Michael Webster ( m.f.webster@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

p-2011-269 – Mathematical Modelling of Polymerising Biological fibers close to Bio-membranes

Set by Dr Rob Daniels ( d.r.daniels@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

Living cells need to form cellular protrusions in order to sense their environment, and migrate. A biological cell achieves this by producing long, thin fibers of actin which go on to push against the cell membrane and/or an underlying cytoskeletal cortex. This project will involve mathematical modelling of the growth of biological fibers which interact with a membrane surface.

p-2011-272 – Mathematical Modelling of an AFM tip

Set by Dr Rob Daniels ( d.r.daniels@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

The Atomic Force Microscope (AFM) is a powerful tool to investigate the structure of many surfaces of interest. The AFM works by bringing its tip very close to a surface and recording its deflection. This project will involve mathematical modelling of the mechanical and dynamical properties of an atomic force microscope cantilever tip, including possible vibrational and/or tapping motion.

p-2011-322 – Monte Carlo simulations of polymer conformations

Set by Dr Anthony Higgins ( a.m.higgins@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

This computational project looks at the physics of long chain polymer molecules. An individual polymer molecule can exhibit conformations ranging from a densely packed globule to a highly extended structure. This project will use numerical methods to simulate polymer conformations and hence calculate the probability distribution function P(R,N) of an isolated polymer chain in solution (P(R,N)= the probability that a polymer of degree of polymerisation N has an end-to-end vector of magnitude R). The project will then look at either i) the effect of polymer-solvent interactions on conformation or ii) simulations of dynamics using either Monte Carlo of molecular dynamics techniques.

p-2011-323 – Numerical simulation of phase separation in binary liquid mixtures

Set by Dr Anthony Higgins ( a.m.higgins@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

Phase separation in binary liquid mixtures occurs when an initially homogeneous system breaks up into two co-existing phases. For example, a mixture of two polymers can undergo separation into two phases (each phase being highly concentrated in only one of the two polymers) in response to a change in temperature. This computational project will use finite-difference methods to simulate the time development of the phase-separating morphology both in liquid mixtures in one dimension (see Ball & Essery J. Phys.; Condens. Matter 2 (1990) 10303).

p-2011-326 – The flow behaviour of conjugated polymer solutions and melts; literature review

Set by Dr Anthony Higgins ( a.m.higgins@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

Conjugated polymers are relatively new materials that consist of alternating double & single carbon-carbon bonds along the backbone of the polymer. This structure leads to the delocalisation of electrons along the polymer backbone. This delocalisation enables electrical conduction to occur within a single polymer molecule and gives rise to the semiconducting & light-emitting properties exhibited by many different conjugated polymers. However, it also greatly increases the stiffness of the polymer chains and is expected to influence the flow behaviour of the polymers when molten.

This project involves an extensive literature survey on the flow properties of both molten conjugated polymers and conjugated polymers in solution. The focus of the survey should be on i) what are the mechanical flow properties (rheology) of these polymers and how do they differ from other polymeric and liquid-crystalline materials? ii) how can conjugated polymers be applied to substrates to make electronic circuits via printing & patterning techniques? iii) can the mechanical flow properties of conjugated polymers be exploited to improve performance in plastic electronic devices such as diodes and transistors?

p-2011-365 – Finite Element Model Updating using Response Surface Method

Set by Dr Rajib Chowdhury ( r.chowdhury@swansea.ac.uk ), Materials Research Centre ( MRC ).

Description

This project proposes the response surface method for finite element model updating. The response surface method is implemented by approximating the finite element model surface response equation by adaptive high dimensional model representation (HDMR). The updated parameters of the finite element model were calculated using genetic algorithm by optimizing the surface response equation. The proposed method will then be compared with the existing methods that use simulated annealing or genetic algorithm together with a full finite element model for finite element model updating. The proposed method will be tested on an aircraft wing structure. It is anticipated that the proposed method will provide the updated natural frequencies and mode shapes that were of the same order of accuracy as those given by simulated annealing and genetic algorithm. Furthermore, the proposed approach would provide an efficient computational tool compared with the genetic algorithm and the simulated annealing.

Special requirements

Matlab, finite element software (ANSYS)
