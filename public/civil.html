p-2011-191 – Finite element modelling in Engineering

Set by Prof Michael Edwards ( m.g.edwards@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

In this project the finite element method will be investigated and applied to fluid flow problems. After a literature review the student will develop FEM in matlab and investigate stability and accuracy issues. Classical convection-diffusion problems will be studied, arising in engineering fluid dynamics with application in aerospace, civil, mechanical and chemical engineering.

p-2011-193 – Biomedical Engineering Methods for Modelling of Virus Spreading

Set by Prof Michael Edwards ( m.g.edwards@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

After a literature review the student will formulate the governing equations and learn how to approximate ordinary differential equations. The project will involve a comparative study of different numerical methods for ordinary differential equations, in order to determine their benefits for biomedical applications.

After a study of the numerical methods, the student will write a Matlab program to solve ordinary differential equation problems arising in biomedical problems. Validation against analytical test solutions will be performed. The methods will then be compared with those available in matlab.

Special requirements

MATLAB (available on University PC’s) £100. library

p-2011-207 – Micro-fluid dynamics simulation: CFD modelling of micro-cavity liquid transfer

Set by Dr Davide Deganello ( d.deganello@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

Liquid transfer from micro-cavities onto a substrate is at the base of several printing processes. The optimisation of this transfer process is crucial for novel applications such as organic electronic, biological printing. The aim of the project is to model the effect of the cavity shape on the liquid transfer process, using an existing numerical simulation model. The student will be provided of the required background information and of the starting numerical model. The numerical side of the project will be integrated with some basic laboratory experiments to provide the student with a more comprehensive understanding of the modelled physical process.

p-2011-237 – Multimedia Film Making for Rheology

Set by Prof Michael Webster ( m.f.webster@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

p-2011-274 – Mathematical Modelling of Cell Motility on a surface

Set by Dr Rob Daniels ( d.r.daniels@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

Living cells need to move around on surfaces in order to carry out biological functions properly. This project will involve mathematically modeling the motion of biological cells on surfaces as a random walk. The influence of the surface properties (such as porosity) on the motion of cells will be investigated theoretically, and compared to pre-existing experimental data.

p-2011-319 – Optimisation of Fourier transform mechanical spectroscopy of biopolymer gelation

Set by Prof Rhodri Williams ( p.r.williams@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

The project will seek to establish the optimum frequency content for accurate detemination of the gel point of collagen solutions by Fourier transform mechanical spectroscopy (FTMS) based on small amplitude oscillatory shear. The appropriate level of strain amplitude associated with each frequency component of the FTMS waveform will be independently assessed in both controlled displacement and controlled torque instruments. The project will also seek to establish the limits of collagen concentration, within the physiologically relevant range, for which viable rheometric protocols can be developed.

p-2011-364 – Correlated function expansion approach for damage identification.

Set by Dr Rajib Chowdhury ( r.chowdhury@swansea.ac.uk ), Materials Research Centre ( MRC ).

Description

A family of multivariate representations is introduced to capture the input–output relationships of high- dimensional physical systems with many input variables. A systematic mapping procedure between the inputs and outputs is prescribed to reveal the hierarchy of correlations amongst the input variables. It is argued that for most well-defined physical systems, only relatively low-order correlations of the input variables are expected to have an impact upon the output. Correlated function expansion (CFE) utilize this property to present an exact hierarchical representation of the physical system. At each new level of CFE, higher-order correlated effects of the input variables are introduced. This methodology has been widely employed in many applications such as uncertainly propagation, design optimization, and response prediction. But so far the literature related to its application in structural damage identification (SDI) is scarce. Therefore this study attempts to present a systematic SDI procedure comprising four sequential steps of feature selection, parameter screening, primary meta-modeling and updating, and reference-state meta-modeling with SDI realization. The last two steps imply the implementation of inverse problems by model updating in which the meta-models substitute the FE models.

Special requirements

Matlab, finite element software (ANSYS)

p-2011-386 – Optimisation

Set by Dr Andrew McCowen ( a.mccowen@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

There are many applications in Engineering were software is used in design or in fundamental research to establish the optimum conditions or optimum performance. When there are many possible variables and the software used in the modelling has a high computational demand, both in cpu and memory, then it is vital to have an optimisation strategy that minimises the number of tests that have to be run to establish optimised conditions or performance. This project will review such strategies and report their performance.

p-2011-448 – An Oyster card for lectures – using RFID to track student attendance

Set by Dr Richard Cobley ( richard.j.cobley@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

This project will develop a system where students use their student card to 'sign in' to a lecture, recording attendance.
University student cards have several methods of identifying themselves.
We could explore those and look at recognising student cards and creating an electronic register as an exercise in designing building access systems that might be taken on as part of a Civil or Mechanical Engineering design project.

The student will either need some understanding of object-orientated programming, or be willing to learn as part of the project.
Some of the development may be in coding the software to drive the card reader, which already exists. However, the brief can be adjusted to suit the interests and skills of a Civil or Mechanical Engineer.
