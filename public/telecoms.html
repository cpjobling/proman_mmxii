p-2011-128 – Next generation Green Access Network

Set by Dr Karin Ennser ( k.ennser@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

The project consists on investigating techniques to reduce the power consumption in Passive Optical Network and Fibre-to-the-Home network. Hardware and software aspects will be investigated and methods to implement an energy-aware network will be proposed.

Associated with: Atkins Carbon Critical (C2)

Cross disciplinary research theme(s): Energy and environment

p-2011-230 – Location Based Services

Set by Dr Rashid Mehmood ( r.mehmood@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

Location Based Services (LBS) and the associated technologies are fundamental to many of the mobiles services that we enjoy today in our daily life. We are developing a framework for location based services on the University campus. There are a number of strands of work that you could involve in from design of a location based service or analysis of location data. The student will join an existing research group working to develop various aspects of location based technologies and services and this project will supplement to the existing research being carried out by the group. You are expected to have fairly good skills and interest in ICT technologies, programming and simulations.

p-2011-240 – Optical Wireless Communication for Indoors

Set by Dr Karin Ennser ( k.ennser@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

The project is related to emerging technology on indoor optical wireless communciation. Several components and system technologies will be investigated. Potential applications for daily life will be assessed for different applications as sensor, data/video transmission and others.

Cross disciplinary research theme(s): Aerospace and manufacturing

Special requirements

Project may need a small budget for small items.
