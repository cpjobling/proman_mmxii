p-2011-030 – Geometrically-Necessary Dislocation Accumulation at Grain Boundaries

Set by Prof Valerie Randle ( v.randle@swansea.ac.uk ), Materials Research Centre ( MRC ).

Description

Geometrically-necessary dislocation (GND) content can be determined by the effect of dislocations on the lattice curvature of a crystal. Local orientations are measured using Electron Backscatter Diffraction (EBSD), which are then used to determine the local orientation gradients and resolved into the GND content. There is experimental evidence that GND content accumulates at grain boundaries during plastic deformation, the magnitude of which may be related to the geometry of the grain boundary. The objective of this project will be to determine any relationship between the GND content accumulated at a grain boundary and the grain boundary character in strained and annealed samples of copper. The student will use orientation data obtained using EBSD to create maps of GND content in the sample. The student will then use image analysis software to obtain plots of GND content as a function of distance from the boundary. Once a sufficient number of boundaries are measured in this manner, the data will be compiled according to the boundary geometry and character as well as the amount of induced strain. This project is also suitable for Materials or Mechanical students.

Special requirements

Electron Backscatter Diffraction, SEM, Induction Furnace, Metallographic Specimen Preparation, Data Analysis, Image Analysis Software

p-2011-031 – Relative grain boundary energies from dihedral angle measurements

Set by Prof Valerie Randle ( v.randle@swansea.ac.uk ), Materials Research Centre ( MRC ).

Description

During long duration, low temperature anneals there is evidence that the dihedral angles around triple junctions can change, allowing the boundaries to lie in lower energy positions. The objective of this project is to devise a method to measure the change in these dihedral angles, for body-centred cubic metals, before and after long duration, low temperature anneals. Conclusions are to then be drawn from the observations in regard to change in dihedral angle and grain boundary energies. This project is also suitable for Materials or Mechanical students.

Special requirements

Electron Backscatter Diffraction, SEM, furnace, metallographic specimen preparation

p-2011-191 – Finite element modelling in Engineering

Set by Prof Michael Edwards ( m.g.edwards@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

In this project the finite element method will be investigated and applied to fluid flow problems. After a literature review the student will develop FEM in matlab and investigate stability and accuracy issues. Classical convection-diffusion problems will be studied, arising in engineering fluid dynamics with application in aerospace, civil, mechanical and chemical engineering.

Special requirements

MATLAB (available on University PC’s) £100. library

p-2011-199 – Finite element analysis of non-local beams and plates

Set by Prof Sondipon Adhikari ( s.adhikari@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

Nanocomposite materials hold significant promise to reduce weight and increase the strength of materials to be used in future aircraft and spacecraft structures. Nanocomposites contain nano particles such as carbon nanotubes or graphenes. Nonlocal model is highly suitable for nano scale systems as opposed to conventional local model. This project will develop a novel finite element analysis for nano materials based on non-local theory.

p-2011-234 – Environmental Impact of Subglacial-ice sheet flow – Rheological Modelling

Set by Prof Michael Webster ( m.f.webster@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

This project will compare and contrast the predictions of various continuum models for flow in substrata of glaciers. A principal aspect of this study will be to investigate the variation of the onset of yield stress and its influence on material system response through generalized Herschel-Bulkley modelling; including shear thinning/thickening behaviour. Alternative modeling approaches shall include viscoplastic (yield stress), viscoelastic and visco-elasto-plastic (yield stress and memory) idealisations. The work will contain a literature review, and introduce front-tracking and slip boundary conditions.

Special requirements

Mathlab, Personal Laptop (would be helpful)

p-2011-334 – Development of Cost Models for Trusses

Set by Prof Johann Sienz ( j.sienz@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

Trusses are widely used in aerospace, mechanical and civil engineering. Optimal design of these trusses can be done using performance criteria, such as stiffness, strength and weight. Although these are valid criteria, cost is becoming an increasingly important factor. Some of the cost is linearly linked to the weight of the structure. Other cost factors are more difficult to quantify. The aim of this project is to develop a cost model for trusses with an emphasis on manufacturing cost, and here by concentrating on the cost of manufacture of joints. The overall cost model is to be implemented in Matlab so that it can be used in in-house truss optimization programmes.

Cross disciplinary research theme(s): Aerospace and manufacturing

Special requirements

Matlab

p-2011-336 – Exploration of the Kreisselmeier-Steinhauser function for Optimization

Set by Prof Johann Sienz ( j.sienz@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

Most problems in structural optimization must be formulated as constrained problems. In a typical structural design problem the objective function is a fairly simple function of the design variables (e.g., weight), but the design has to satisfy a host of stress, displacement, buckling, and frequency constraints. These constraints are usually complex functions of the design variables available only from an analysis of a ﬁnite element model of the structure. The objective of this work is to explore the features of the Kreisselmeier-Steinhauser function and investigate the usability of this special function in optimization. The project will involve extensive programming in Matlab.

Cross disciplinary research theme(s): Aerospace and manufacturing

Special requirements

Matlab

p-2011-388 – Microwave printed antenna for circular polarisation

Set by Dr Andrew McCowen ( a.mccowen@swansea.ac.uk ), Multidisciplinary Nanotechnology Centre ( MNC ).

Description

There have been some interesting structures proposed recently for producing circular polarisation from printed antenna. The successful operation of such structures would be particularly important as microwave antennas for onboard satellite systems. This project will initially test an existing design using an antenna simulator and then develop a procedure to improve its performance.

p-2011-407 – Review of time series models for aerospace applications

Set by Dr Rajesh Ransing ( r.s.ransing@swansea.ac.uk ), Civil and Computational Engineering Centre ( C2EC ).

Description

A typical aircraft is full of sensors and measurement devices that record every aspect of a flight every second. Sometimes this data is monitored live by engine manufacturers. The main objective is to discover abnormal behaviour of systems even though its occurrence is very rare.

This project will review challenges in data measurement and capture along with time series models that are used to make sense from this data. The student also has an opportunity to develop some of the simple models in MATLAB to enhance his/her understanding in this subject. The technical skills developed in this project are generic and can be applied to number of other fields.

Cross disciplinary research theme(s): Aerospace and manufacturing

Special requirements

Interlibrary loans/books
